# Sequência de Emails — Onboarding (Novo Cliente)

> **Público:** CS / Automação de onboarding  
> **Objetivo:** Levar o cliente ao primeiro valor real em 7 dias  
> **Data:** 2026-05-30  
> **Gatilho:** Enviado a partir do momento de criação da conta

---

## Filosofia do onboarding Ekthos

O pastor não tem tempo para tutoriais longos. O objetivo de cada email é uma ação específica — não informar, mas mover. Cada email resolve um obstáculo, não descreve um recurso.

**Sequência de ativações (em ordem de valor):**
1. WhatsApp conectado → Agente responde
2. Membros importados → Pipeline visível
3. Primeiro handoff real → Líder encantado
4. Relatório gerado → Pastor viu o valor completo

**Gatilho de envio:** WhatsApp conectado
